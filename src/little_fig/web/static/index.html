📖 Little Fig Docs

v0.5 · Fig Engine

🚀 Quick Start

  1. Enter a model in the Chat sidebar (e.g. Qwen/Qwen2.5-0.5B-Instruct)
  2. Click Load — wait for status
  3. Type a message and press Enter

🔧 Components

FigQuant

Adaptive codebook INT4. 9.7% less error than standard.

FigKernel

torch.compile fused ops. 2.95× faster RMSNorm.

FigPipeline

Async GPU-CPU training. Optimizer on CPU.

Universal GGUF

Auto-detects any architecture. 3-strategy loader.

🐛 Troubleshooting

  • Model won't load? pip install "transformers>=5.5.0"
  • OOM? Use smaller model or free RAM first
  • Gemma 4? Needs transformers ≥5.5.0 (Strategy 3 direct loading)
