🚀 Quick Start

  1. Enter a model ID in the sidebar (e.g. Qwen/Qwen2.5-0.5B-Instruct) or browse for a local .gguf file
  2. Click Load and wait for the green status pill
  3. Type a message and press Enter — responses stream in real-time
  4. Attach files (📎), use voice input (🎤), or paste images

🔧 Fig Engine Components

FigQuant

Adaptive codebook INT4. NF4 base refined via weighted k-means. 9.7% less error than standard INT4.

FigKernel

torch.compile fused ops. RMSNorm 2.95× faster. Works on CPU (AVX-512) and GPU (CUDA).

FigPipeline

Async GPU-CPU training. Optimizer states on CPU. Train models larger than VRAM.

Universal GGUF

Auto-detects any architecture. 22+ via transformers, 122+ via gguf-py fallback.

🐛 Troubleshooting

  • Model won't load? pip install --upgrade transformers gguf
  • Out of memory? Use GPT-2 or Qwen 0.5B. Free RAM before loading.
  • Gemma 4? pip install git+https://github.com/huggingface/transformers.git
